  • Создание патч листа
  • Регистрация в Лаунчере

В первом поле укажите папку где находится игровой клиент, который необходимо запатчить.

В следующем поле укажите папку сохранения архивов.

После завершения архивации, в папке сохрания появится файл patch.json, это файл с данными вашего патча.

Файл patch.json и все файлы необходимо загрузить на ваш сервер.

Далее, при регистрации вашего сервера в базе лаунчера, необходимо указать прямые ссылки на patch.json и архивы.

Статус Ожидание

Все сервера отображаемые в лаунчере, находится в общей базе, при регистрации, вы добавляете свой сервер в базу.

Вам необходимо заполнить данные необходмые для лаунчера.

После заполнения, вы получите код ваших настроек, его нужно передать разработчику лаунчера.

После успешной проверки, ваш сервер появится в базах у всех пользователей лаунчера.

Проверка владельца сайта.
Для того чтоб ваш сервер попал в базу лаунчера, необходимо проверить, является сайт вашим.
В корне вашего сайта, создайте пустой файл {{ .CreateCode }}
Пример: https://example.com/{{ .CreateCode }}
Если на момент проверки файла не будет, регистрация будет отклонена.
